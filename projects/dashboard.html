Свое.Шеф —
Дашборд сезонности

Задача

Создать дашборд, который помогает ресторанам и шеф-поварам видеть сезонность продуктов, анализировать меню и получать рекомендации по обновлению ассортимента с учётом типа кухни и текущего сезона.

Роль

User Flow • UI Design • Prototyping • Дизайн-ревью

Год

2026

Продукт

Сервис Свое.Шеф (фермерские продукты для HoReCa)

User Flow Основной сценарий работы с дашбордом: загрузка меню → анализ сезонности → получение рекомендаций по замене продуктов

Исследование Выявленные болевые точки шеф-поваров: сложность отслеживания сезонных продуктов, высокая зависимость от поставщиков и отсутствие прозрачной аналитики

Wireframes Низкофидельные версии основных экранов дашборда

Главный дашборд Обзор сезонности продуктов и рекомендации по обновлению меню

Детальный анализ Карточки продуктов с информацией о сезонности, цене и доступности у фермеров

Интерактивный прототип Полноценный кликабельный прототип в Figma (можно вставить ссылку)
